Question 1: Function Testing Basics

Write a function called celsius_to_kelvin() that converts temperatures from Celsius to Kelvin (add 273.15). Then write TWO tests using testthat to verify:

  1. Water freezes at 273.15 K (0°C)
  2. Water boils at 373.15 K (100°C)
Hint: Use expect_equal() in your tests. Remember the formula: K = C + 273.15
Solution:
celsius_to_kelvin <- function(temp) { temp + 273.15 } test_that("Water freezes at 273.15 K", { expect_equal(celsius_to_kelvin(0), 273.15) }) test_that("Water boils at 373.15 K", { expect_equal(celsius_to_kelvin(100), 373.15) })

Question 4: Multi-Stage Simulation

Food delivery: Orders ~ Poisson(50), Prep ~ Exp(20), Delivery ~ Log-Normal(3, 0.5)

Simulate 1000 days and estimate P(total time > 2000 min)

Solution:
simulate_day <- function() { n_orders <- rpois(1, 50) prep_times <- rexp(n_orders, rate = 1/20) delivery_times <- rlnorm(n_orders, 3, sqrt(0.5)) sum(prep_times + delivery_times) } results <- replicate(1000, simulate_day()) prob_exceed <- mean(results > 2000)

# Analytical: E[Y] = E[N] × E[M] × E[X] # = 8 × (1/0.3) × 150 = 4000 bytes/min # Simulation verification simulate_minute <- function() { n_bursts <- rpois(1, 8) total <- 0 for(i in 1:n_bursts) { n_packets <- rgeom(1, 0.3) + 1 total <- total + sum(rexp(n_packets, 1/150)) } total } mean(replicate(10000, simulate_minute()))

Create safe_divide_lists() with error handling, *args/**kwargs, and multiple return values.

Solution:
def safe_divide_lists(numerators, denominators, default_value='skip', **kwargs): results, errors = [], [] for i, (n, d) in enumerate(zip(numerators, denominators)): try: if d == 0: if default_value == 'skip': errors.append(f"Div by zero at {i}") continue results.append(default_value) else: results.append(n / d) except Exception as e: errors.append(f"Error at {i}: {e}") return results, errors
